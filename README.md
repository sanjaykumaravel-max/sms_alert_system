# SMS Alert App\n\nRun steps:\n1. Create and activate venv: `python -m venv .venv` then `.venv\Scripts\activate`\n2. Install deps: `pip install -r requirements.txt`\n3. Add Fast2SMS API key to .env: `SMS_API_KEY=your_key_here`\n4. Run: `python src/main.py`\n\nDev notes:\n- Toggle SMS sending in Settings.\n- Test SMS via Settings -> Send Test SMS.\n- Logs written to `logs/app.log`.\n
